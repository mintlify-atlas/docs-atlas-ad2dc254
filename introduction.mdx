## What is Vitacore?

Vitacore is a clinical management platform built for healthcare providers. It centralizes patient records, clinical documentation, and administrative configuration in a single web application — enabling medical teams to focus on patient care rather than paperwork.

Built with **Flask** and **Supabase**, Vitacore provides a secure, role-controlled environment where every user action is governed by a configurable permission matrix.

## How it works

Vitacore is organized around a central patient chart. From the patient detail view, clinical staff can access all clinical history, record new evolution notes, and log vital signs. Administrative users can configure the organizational structure — headquarters, offices, specialties, and services — while system administrators manage user accounts and access permissions.
